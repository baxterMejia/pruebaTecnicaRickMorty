# Rick and Morty Human Detector

Este proyecto identifica a los personajes humanos de Rick and Morty. Creado con Next.js en el frontend, Express en el backend y utilizando la API GraphQL de Rick and Morty.

## Cómo correr el proyecto:

### Backend:
1. Dirígete a la carpeta `/backend`.
2. Ejecuta `npm install` para instalar las dependencias.
3. Ejecuta `node server.js` para iniciar el servidor en el puerto 4000.

### Frontend:
1. Desde la raíz, ejecuta `npm install`.
2. Luego, ejecuta `npm run dev` para iniciar el servidor de desarrollo.

## Características:
- Lista de personajes humanos.
- Paginación (si decides implementarla).
- Diseño con Material-UI.
